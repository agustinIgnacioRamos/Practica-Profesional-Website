Resumen del Proyecto

Surge para solucionar problemas típicos como la aglomeración en los accesos, la desinformación sobre precios y las dificultades de navegación dentro del predio.

El proyecto CarnvalPass abarca el desarrollo de una aplicación para los espectadores del Carnaval de Gualeguaychú.

La metodología de trabajo adoptada es ágil, énfasis en la comunicación y entrega rápida. Además, se utiliza el paradigma de orientación a objetos para estructurar el software y UML para el modelado.

La aplicación busca gestionar accesos, comprar tickets, mostrar ágilmente información y compartir precios, mejorando de esta manera la experiencia del usuario y la organización general del evento. .
